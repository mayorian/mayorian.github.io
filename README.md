# ⚡ JSmayo

A lightweight collection of browser-based JavaScript utilities and small experiments presented in a clean dashboard interface.

JSmayo is a fast, dependency-free playground where every tool runs directly in the browser — no installation, no backend, and no build step.

---

## 📌 Overview
- 🟡 Plain JavaScript (no frameworks)
- 🎨 Responsive Bootstrap UI
- 🔎 Searchable tool launcher
- 🧩 Modular and readable code
- 🌐 GitHub Pages ready

---

## 🧰 Included Utilities
- 🧠 JavaScript syntax analyzer
- 📱 QR code generator
- 🪧 Destination display board simulation
- 🔁 Form and input interaction demos
- ✅ Prompt validation tool
- 🎓 Study / practice helpers

---

## ▶️ Running the Project
Simply open:

`index.html`

The project is fully static and works on any static hosting.

---

## 🎯 Purpose
JSmayo acts as:
- 🧪 a small utility toolkit
- 📘 a learning reference
- 🎨 a UI interaction showcase

Focused on simplicity, clarity, and practical examples rather than large application architecture.
