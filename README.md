# 🌌 JSmayo — Futuristic JavaScript Micro-Tools Hub

**Not a framework.
Not a template.
A playground.**

Welcome to **JSmayo**, a neon-styled laboratory of small, fast, useful JavaScript tools — all running directly in the browser.

No installs.
No backend.
No build process.

Just open and use.

---

## ⚡ Why this exists

Modern web tools are often:

* heavy
* over-engineered
* dependent on 50+ packages

**JSmayo does the opposite.**

It proves you can still build powerful interactive utilities using:

* 🟡 Plain JavaScript
* 🎨 Smart CSS
* 🧩 Clean structure

The goal is simple:

> Instant tools. Instant understanding. Instant learning.

---

## 🧠 What you’ll find inside

A collection of small but practical browser utilities:

🧪 Syntax analyzers
📱 QR generators
🪧 Simulation boards

🤖 Prompt validation tools
🎓 Study helpers

Each tool is:

* isolated
