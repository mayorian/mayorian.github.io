ServiceNow Query API Guide

GlideRecord

Classic API for full CRUD operations. Best for inserting, updating, deleting, and complex record workflows. 
var gr = new GlideRecord('incident');
gr.addActiveQuery();
gr.addQuery('priority', 1);
gr.query();

while (gr.next()) {
  gs.info(gr.number);
}

GlideQuery

Modern fluent API for clean and readable read-only queries. Ideal when selecting specific fields. 
var results = new GlideQuery('incident')
  .where('active', true)
  .select(['number','priority'])
  .limit(10)
  .toArray();

results.forEach(function(r){
  gs.info(r.number);
});

GlideAggregate

Optimized for calculations like COUNT, SUM, AVG and grouped results. Best for dashboards and reporting logic. 
var ga = new GlideAggregate('incident');
ga.addActiveQuery();
ga.addAggregate('COUNT');
ga.groupBy('priority');
ga.query();

while (ga.next()) {
  gs.info("Priority " + ga.priority +
          ": " + ga.getAggregate('COUNT'));
}

Main Difference: GlideQuery vs GlideAggregate

🟠 GlideQuery = “Give me records.”
Returns actual rows from the database.
🟢 GlideAggregate = “Give me numbers about records.”
Returns grouped metrics and calculations.
Performance Tip:

❌ Inefficient counting: 
var count = new GlideQuery('incident')
  .where('active', true)
  .toArray()
  .length;
✔ Efficient counting: 
var ga = new GlideAggregate('incident');
ga.addActiveQuery();
ga.addAggregate('COUNT');
ga.query();

if (ga.next()) {
  gs.info(ga.getAggregate('COUNT'));
}
